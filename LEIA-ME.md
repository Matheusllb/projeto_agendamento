# Manual da Raiz do Projeto

Esta pasta contém a estrutura base de todo o projeto (Frontend e Backend) e arquivos de configuração global.

## Arquivos Importantes

- **angular.json**: Arquivo de configuração principal do Angular CLI. Define como o projeto é construído, testado e servido.
- **package.json**: Lista todas as dependências (bibliotecas) do projeto frontend e scripts de execução (como `npm start`).
- **tailwind.config.js**: Configuração do framework de CSS Tailwind. Aqui definimos cores personalizadas, fontes e extensões do tema.
- **tsconfig.json**: Configurações do compilador TypeScript.
- **README.md**: Documentação geral do projeto.

## Pastas
- **src/**: Contém todo o código fonte do Frontend (Angular).
- **backend/**: Contém o código fonte do servidor (API).
