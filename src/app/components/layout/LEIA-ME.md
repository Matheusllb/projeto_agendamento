# Manual de Layout

Esta pasta contém os componentes estruturais que formam o esqueleto da aplicação.

## Conteúdo Típico
- **Header**: Barra superior com navegação e perfil.
- **Sidebar**: Menu lateral.
- **Footer**: Rodapé.
- **MainLayout**: O componente que junta tudo isso e define onde o conteúdo da página vai aparecer.
