# Manual de Componentes

Esta pasta contém componentes "burros" (dumb components) ou de apresentação, que são focados em UI e podem ser reutilizados em várias partes do sistema.

## O que guardar aqui?
- Botões personalizados.
- Campos de formulário.
- Modais e Popups.
- Layouts (Header, Sidebar, Footer).

## Regra de Ouro
Componentes aqui devem ser o mais genéricos possível para facilitar o reuso.
