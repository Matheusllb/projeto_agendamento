# Manual da Pasta Source (src)

Esta é a pasta principal de desenvolvimento do Frontend. Tudo que é compilado e enviado para o navegador do usuário vive aqui.

## Arquivos e Pastas
- **app/**: O coração da aplicação. Contém componentes, serviços e lógica.
- **assets/**: Arquivos estáticos como imagens, ícones e fontes locais.
- **environments/**: Configurações variáveis por ambiente (Desenvolvimento vs Produção).
- **index.html**: O arquivo HTML base que carrega a aplicação Angular.
- **main.ts**: O ponto de entrada do JavaScript que inicializa o Angular.
- **styles.css**: Estilos globais e importação do Tailwind CSS.
